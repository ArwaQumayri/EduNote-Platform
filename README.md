# EduNote Platform

EduNote is a simple web-based platform that helps university students share and access study notes easily. The project was developed as part of **COMP315 – Web Programming** at **Jazan University**.

## Features
- Create an account and log in.
- Upload PDF notes categorized by college.
- Clean and student-friendly interface.
- View and download uploaded notes.
- LocalStorage used for storing notes (for demonstration).
- PHP and MySQL used for managing user accounts.

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  
- LocalStorage
